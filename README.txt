WORK2WISH DASHBOARD - INLINE DEMO VIDEO
======================================

1. Extract this ZIP folder.
2. Double-click start_dashboard.bat.
3. The dashboard opens automatically in your browser.
4. Open the About page and play the demo video inside the dashboard.
5. Keep the black PowerShell window open while using the dashboard.
6. Press Ctrl+C in that window when finished.

For Vercel or normal web hosting, upload index.html as usual.
